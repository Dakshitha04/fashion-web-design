# 👗 Deerthi Fashions – Frontend Fashion Website

A modern and responsive fashion website designed using **HTML5** and **CSS3**. This project showcases a stylish fashion brand website with multiple pages and an elegant user interface.


## 📌 Project Overview

Deerthi Fashions is a frontend web development mini project created to demonstrate the fundamentals of HTML and CSS. The website provides an attractive and user-friendly interface for a fashion brand, including product collections, company information, fashion blogs, and a contact page.


## ✨ Features

- Responsive homepage
- Attractive hero section
- Fashion collections page
- About Us page
- Fashion blog page
- Contact form
- Clean and modern UI
- Easy navigation between pages


## 🛠️ Technologies Used

- HTML5
- CSS3

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/deerthi-fashions-frontend.git
```

2. Open the project folder.

3. Open `index.html` in any web browser.

No additional installation or dependencies are required.
